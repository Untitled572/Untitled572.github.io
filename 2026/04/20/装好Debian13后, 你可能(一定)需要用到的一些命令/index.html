装好Debian13后, 你可能(一定)需要用到的一些命令

发表于 2026-04-20 更新于 2026-04-22

众所周知, Debian向来以对新手不友好而著称, 笔者本人作为小白, 试过了几个不同的发行版后, 仍然选择了Debian作为主力系统, 下面介绍几个个人总结的常用（必用）命令

Sudo权限

安装过程中, 如果设置了管理员密码, 则后续设置的用户默认是没有sudo权限的, 可以通过以下命令来赋予权限

中文界面

安装时候最好用英文安装, 中文的话文件夹是中文名, 安装后再更改语言设置

随后翻到最下面找到zh_CN.UTF-8 UTF-8, 使用空格键选中后, 按回车键下一步, 选中zh_CN.UTF-8作为默认区域设置

挂载硬盘

首先使用df -h查看当前已经挂载的分区, 随后使用lsblk查看所有硬盘及分区

切换GUI/CLI

如果不需要显示图形化界面, 配置完成后可以切换到命令行界面来释放图形界面占用的资源

更改系统启动顺序

Windows 11的引导只能进入Windows系统, 而Debian的GRUB默认进入Debian系统, 可以修改GRUB使得bios引导GRUB后默认进入Windows

找到以下行

GRUB_DEFAULT=0

编辑该行，将 0 替换为您要默认启动的项目编号。就我而言，我想将默认启动更改为 Windows。因此我设置了第三项2（因为计数从0开始）。

GRUB_DEFAULT=2

随后使用sudo update-grub重新加载grub
