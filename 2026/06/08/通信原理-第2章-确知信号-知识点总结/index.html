通信原理 第2章 确知信号 知识点总结

发表于 2026-06-08 更新于 2026-06-09

知识点总结, 期末复习用, AI生成

2.1 确知信号的类型

  • 确知信号的定义：在定义域内的任意时刻都有确定的函数值。反之则为随机信号或不确知信号。
  • 按周期重复性分类：
    • 周期信号：每隔一定的时间间隔按相同规律重复，且无始无终。满足 $s(t) = s(t + T_0)$ 的最小 $T_0 > 0$ 称为信号的基波周期。
    • 非周期信号：不具备周期重复性的信号。
  • 按信号能量是否有限分类：
    • 能量信号：若信号的能量 $E$ 满足 $0 < E < \infty$ 且其平均功率 $P \to 0$，则称其为能量（有限）信号（如单个矩形脉冲）。
    • 功率信号：若信号的平均功率 $P$ 满足 $0 < P < \infty$ 且其能量 $E \to \infty$，则称其为功率（有限）信号（如直流信号、周期信号 and 随机信号）。
    • 注意：一个信号不可能既是能量信号又是功率信号。

2.2 确知信号的频域性质

信号的频域性质反映了信号各频率分量的分布情况，涉及占用的频带宽度、滤波性能等，是信号最重要、最本质的性质之一。

2.2.1 功率信号的频谱（周期信号）

对于周期为 $T_0$ 的功率信号 $s(t)$，可展开为指数型傅里叶级数：
$$s(t) = \sum_{n=-\infty}^{\infty} C_n e^{j 2 \pi n f_0 t}$$
其中，傅里叶级数系数（又称双边谱）为：
$$C_n = C(n f_0) = \frac{1}{T_0} \int_{-T_0/2}^{T_0/2} s(t) e^{-j 2 \pi n f_0 t} dt$$

  • $f_0 = 1/T_0$ 称为信号的基频， $n f_0$ 称为信号的**$n$次谐波频率**。
  • $|C_n|$ 随频率变化的特性称为幅度谱（其值为单边谱振幅的一半）。
  • $\theta_n$ 随频率变化的特性称为相位谱。
  • 当 $n=0$ 时，$C_0$ 表示信号的时间平均值，即直流分量。
  • 实信号频谱的性质：对于物理可实现的实信号，正频率部分和负频率部分间存在复数共轭关系，即 $|C_n|$ 满足模偶对称，$\theta_n$ 满足相位奇对称。
  • 若 $s(t)$ 是实偶信号，则 $C_n$ 为实函数；若 $s(t)$ 不是偶信号，则 $C_n$ 为复函数。

2.2.2 能量信号的频谱密度

  • 定义：能量信号 $s(t)$ 的傅里叶变换即为频谱密度 $S(f)$：
    $$S(f) = \int_{-\infty}^{\infty} s(t) e^{-j 2 \pi f t} dt$$
  • $S(f)$ 与 $C_n$ 的主要区别：$S(f)$ 是连续谱，单位是 $\text{V/Hz}$；$C_n$ 是离散谱，单位是 $\text{V}$。
  • 重要物理特例：
    • 单位门函数（矩形脉冲）：其频谱密度为抽样函数（$\text{Sa}$ 函数）形式，且矩形脉冲的带宽等于其脉冲持续时间 $\tau$ 的倒数，即 $\frac{1}{\tau}\text{ Hz}$。
    • 单位冲激函数（$\delta$ 函数）：频谱密度 $S(f) = 1$（全频域均匀分布）。
    • 冲激函数的引入：利用冲激函数，可以把频谱密度的概念推广到功率信号上（例如：无限长余弦波 $\cos(2\pi f_0 t)$ 的频谱密度为 $\frac{1}{2}[\delta(f-f_0) + \delta(f+f_0)]$）。

2.2.3 能量谱密度与功率谱密度

  • 能量谱密度 $G(f)$：
    • 定义：$G(f) = |S(f)|^2$，用来描述能量信号的能量在频域上的分布情况。
    • Parseval定理：信号的总能量既可以在时域求解，也可以在频域对能量谱密度积分求解。
  • 功率谱密度 $P(f)$：
    • 定义：用来描述功率信号的功率在频域上的分布情况。
    • 周期信号的功率谱密度：由各次谐波功率的冲激序列组成：
      $$P(f) = \sum_{n=-\infty}^{\infty} |C_n|^2 \delta(f - n f_0)$$
    • 其中，$|C_n|^2$ 为第 $n$ 次谐波的功率。

2.3 确知信号的时域性质

确知信号的时域性质可由自相关函数或互相关函数来描述。

2.3.1 自相关函数

1、能量信号的自相关函数

  • 定义：$R(\tau) = \int_{-\infty}^{\infty} s(t) s(t + \tau) dt$
  • 性质：
    • 与时间 $t$ 无关，只与时间差 $\tau$ 有关。
    • 当 $\tau = 0$ 时，$R(0) = E$（等于信号的总能量）。
    • 关于 $\tau$ 是偶函数：$R(\tau) = R(-\tau)$。
    • 自相关函数 $R(\tau)$ 和其能量谱密度 $|S(f)|^2$ 是一对傅里叶变换对。

2、功率信号的自相关函数

  • 定义：$R(\tau) = \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{T/2} s(t) s(t + \tau) dt$
  • 性质：
    • 当 $\tau = 0$ 时，$R(0) = P$（等于信号的平均功率）。
    • 同样满足关于 $\tau$ 的偶对称性。
    • 功率信号的自相关函数 $R(\tau)$ 和其功率谱密度 $P(f)$ 是一对傅里叶变换对。
    • 典型结论：周期性余弦信号 $s(t) = A\cos(\omega_0 t + \theta)$ 的自相关函数为 $R(\tau) = \frac{A^2}{2}\cos\omega_0 \tau$，其与初始相位 $\theta$ 无关。

2.3.2 互相关函数

1、能量信号的互相关函数

  • 定义：$R_{12}(\tau) = \int_{-\infty}^{\infty} s_1(t) s_2(t + \tau) dt$
  • 性质：
    • 只与时间差 $\tau$ 有关。
    • 与两个信号相乘的前后次序有关，满足：$R_{12}(\tau) = R_{21}(-\tau)$。
    • 互相关函数 $R_{12}(\tau)$ 和互能量谱密度 $S_{12}(f)$ 是一对傅里叶变换对。

2、功率信号的互相关函数

  • 定义：$R_{12}(\tau) = \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{T/2} s_1(t) s_2(t + \tau) dt$
  • 性质：
    • 若两个周期性功率信号的周期相同，其互相关函数可在单个周期内积分求解。
    • 相乘次序改变时满足：$R_{12}(\tau) = R_{21}(-\tau)$。
    • 互相关函数 $R_{12}(\tau)$ 和互功率谱 $C_{12}(f)$ 是一对傅里叶变换对。
